
# 🐾 Wildlife Animal Detection Web App

An AI-powered web application that detects and classifies wildlife animals from uploaded images using Computer Vision and Deep Learning.

---

# 📌 Features

- Upload animal images
- AI-based animal detection
- Fast prediction results
- Simple and user-friendly interface
- Computer Vision based image classification
- Frontend and backend integration

---

# 🛠 Technologies Used

## Frontend
- HTML
- CSS
- JavaScript

## Backend
- Python
- FastAPI

## AI / Computer Vision
- PyTorch
- Torchvision
- Pretrained Deep Learning Model

---

---

# ⚙️ How It Works

1. User uploads an animal image.
2. The frontend sends the image to the backend.
3. The AI model processes the image.
4. The system predicts the animal class.
5. The result is displayed on the webpage.

---

# 🚀 Installation Guide

## Step 1: Clone Repository

```bash
git clone https://github.com/your-username/wildlife-animal-detection-webapp.git
```

---

## Step 2: Open Project Folder

```bash
cd wildlife-animal-detection-webapp
```

---

## Step 3: Install Requirements

```bash
pip install fastapi uvicorn torch torchvision pillow python-multipart
```

---

## Step 4: Run Backend Server

```bash
uvicorn main:app --reload
```

---

# ▶️ Run the Project

1. Start backend server.
2. Open frontend `index.html` file.
3. Upload animal image.
4. View prediction result.
